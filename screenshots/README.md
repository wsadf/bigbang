# Screenshots

Esta pasta contém screenshots da aplicação funcionando em diferentes dispositivos e resoluções.

## Estrutura Sugerida

- `desktop.png` - Screenshot da versão desktop
- `tablet.png` - Screenshot da versão tablet
- `mobile.png` - Screenshot da versão mobile
- `dashboard-completo.png` - Screenshot do dashboard completo

## Como Adicionar Screenshots

1. Tire screenshots da aplicação rodando em diferentes dispositivos
2. Salve os arquivos nesta pasta com nomes descritivos
3. Faça commit das imagens no repositório
